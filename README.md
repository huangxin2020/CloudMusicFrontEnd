## 为微信云音乐端小程序而开发的管理系统页面

### 线上体验地址 - http://www.cloudmusic.hxin2020.top/

前端的页面使用了Vue-cli脚手架进行的项目搭建 页面UI使用Element-UI组件库

结合后端Node的Koa2写的的服务端实现了与小程序端的相互连接
